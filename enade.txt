 a) Fundamentos de Programação e Linguagens de Programação
                - Tópico 1: Resolução Algorítmica de Problemas; Desenvolvimento de
Programas; Recursividade; Testes de programas
-------------------------------------------------------------------------------------
Considere as afirmações abaixo:

I - Toda função que pode ser produzida por um computador pode ser escrita como função recursiva sem o uso de iteração.

II - Toda função recursiva pode ser descrita sem recursividade via iterações sucessivas.

III - Nenhuma função que pude ser escrita como função recursiva pode ser descrita via interações sucessivasb

IV - Nem toda função que pode ser escrita como função recursiva pode ser descrita via interações sucessivas

A - Somente a afirmação I é verdadeira
B - Somente a afirmação II é verdadeira
C - Somente a afirmação III é verdadeira 
D - Somente a afirmação IV é verdadeira 
E - Apenas as afirmações I e II são verdadeiras (OK)

- Tópico 2: Linguagens de Programação; Paradigmas/Modelos de
Linguagens de Programação; Metodologias de Desenvolvimento de
Programas


Toda linguagem de programação procedural  é imperativa.
Toda linguagem de programação orientada a objetos é imperativa.
Toda linguagem de programação orientada a objetos também é procedural
Nem toda linguagem de programação orientada a objetos é imperativa

A - Somente a afirmação III é verdadeira
B - Apenas as afirmações I e II são verdadeiras (OK)
C - Exceto a afirmação IV, todas as afirmações são verdadeiras
D - Apenas as afirmações II e III são verdadeiras
E - Apenas as afirmações III e IV são verdadeiras

b) Modelos de Linguagens de Programação
                - Tópico 3: Linguagens imperativas, Linguagens declarativas,
Linguagens Funcionais, Linguagens orientadas a objetos

pai(abel,bernardo). 
pai(abel,bia). 
mae(ana,bernardo). 
mae(ana,bia). 
 
parenteSimples(X,Y) :- pai(X,Y). 
parenteSimples(X,Y) :- mae(X,Y). 
 
irmao(X,Y) :- parenteSimples(Z,X),parenteSimples(Z,Y),X\=Y.

irmao(X,Y).
